## 使用方法

1. `.env` ファイルの `SESSIONKEY` に自分の session_key を入力してください  

   例:SESSIONKEY=key000000000000000

2. 人狼サイトに入室した状態で `fool.exe` を起動してください  
3. 起動完了です

---

## session_key の取得方法

1. Chrome Web Store から **EditThisCookie (V3)** をインストール  
2. 人狼サイトを開いた状態で拡張機能を起動  
3. `session_key` が2つ表示されるので、**上側**を使用してください

※ Chromiumベースのブラウザが必要です  
※ exeファイルを実行できる環境を用意してください

## コマンド一覧

※ `""` 内がコマンド  
※ `{}` はパラメータ指定

- `"深淵を唱える...答えよ"`  
→ botが返事をします

- `"深淵を唱える...時を奏でろ"`  
→ 現在時刻を表示します

- `"深淵を唱える...星となれ、{tName}"`  
→ 対象を kick します

- `"深淵を唱える...海は揺蕩い地は揺れる"`  
→ 1〜120行の掃除を行います

- `"深淵を唱える...幕を閉じよ"`  
→ botを終了します

- `"深淵を唱える...言葉を紡げ"`  
→ 厨二病発言を行います

- `"深淵を唱える...淫らな夜を渇望す"`  
→ えちちな発言を行います

- `"深淵を唱える...煌めく夜に乾杯"`  
→ botのステータスを表示します

- `"深淵を唱える...恐れよ"`  
→ 作成者を称える発言を行います

- `"深淵を唱える...羅針盤が指し示す{int|x>1}つの星は{int|x>0}回煌めく"`  
→ ダイスロールを行います

- `"深淵を唱える..."`  
→ ランダムでコマンドを実行します
